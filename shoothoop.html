Shoot Hoop

Summary

This is a VR Game created for Oculus Rift and can also be played using keyboard for Windows Operating System. C# is used for scripting and this game is created using Unity Game Engine. This is a ranking based game depending on the highest score. Objective of the game is to score as many baskets as possible within 60 seconds.
