Block Breaker

Summary

This is a 2D arcade game made with Unity Game Engine. It is a multi-level game and also a ranking based game depending upon the fastest time to complete all three levels of the game. C# is used for scripting. Objective of the game is to destroy all the blocks while not letting the ball fall down. It's also the first game that I ever made using Unity!
